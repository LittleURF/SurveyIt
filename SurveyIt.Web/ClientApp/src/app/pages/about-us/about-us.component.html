Survey It

Hi there! i don't know what cursed path led you to this place, but if you're already here i guess I can say some stuff about it.

This web application is based on ASP .Net core for the back-end, and Angular for what you're looking at right about now. Unless you're not looking :<

It lets you create, fill out or review anonymous surveys. Multiple question types, any answer you can think of and results displayed in an awesome, but still easy to understand way.
